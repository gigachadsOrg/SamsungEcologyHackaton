EcoData

Приложение, которое будет собирать данные для коллективных сервисов. Продукт для коммерческих компаний, который поможет повысить популярность самой компании (маркетинговый ход в сторону экологичности) и привлечь новых клиентов

ПУТЕШЕСТВИЕ

Проблема: отсутствия бизнес-применения сервисов, которые остаются нишевыми и не привлекают большого количества участников

Актуальность: Затрудняется масштабный сбор данных о загрязнении окружающей среды и препятствует эффективной борьбе с экологическими проблемами для коллективных сервисом по экологии
